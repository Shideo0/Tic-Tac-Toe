# Tic-Tac-Toe  

Classic game with Tkinter GUI  

## Installation  
1. Install Poetry (if not installed):  
   ```bash
   pip install poetry
   ```
2. Set up the project:  
   ```bash
   poetry install
   ```

## How to Run  
```bash
poetry run python tictactoe.py
```
